## Getting Started

### Install from source

First, clone the project:

```bash
$ git clone https://github.com/davezuko/react-redux-starter-kit.git <my-project-name>
$ cd <my-project-name>
$ npm i
```

To start the project:

```bash
$ npm run start
```

To launche the tests:

```bash
$ npm run start
```
